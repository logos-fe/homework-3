In programming languages,a closure is a technique for implenting lexically scoped name binding in a language with first-class functions. Operationally, a closure is a record storing a function together with an environment.

У мовах програмування замикання — це техніка для впровадження зв’язування імен із лексичною областю у мові з першокласними функціями. Операційно закриття — це запис, що зберігає функцію разом із середовищем.

В языках программирования замыкание — это метод реализации связывания имен с лексической областью действия в языке с функциями первого класса. С точки зрения эксплуатации замыкание — это запись, хранящая функцию вместе с окружением.
