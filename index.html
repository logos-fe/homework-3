In programming languages, a closure is a technique for implementing lexically scoped name binding in a language with first-class functions. Operationally, a closure is a record storing a function together with an environment.
У мовах програмування замикання є технікою для реалізація зв’язування імен із лексичною областю у мові з першокласні функції. Оперативно закриття є рекордом збереження функції разом із середовищем.
В языках программирования замыкание — это способ реализация связывания имен с лексической областью видимости на языке с функции первого класса. В оперативном отношении закрытие является записью хранение функции вместе с окружением.
